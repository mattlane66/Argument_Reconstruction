# Reconstruct Charity

Use this skill before critique. The goal is to state the argument so clearly and fairly that the original speaker would likely agree with it.

## Procedure

1. Identify the speaker's central claim.
2. Remove needless hostility, exaggeration, or confusion without changing the claim.
3. Define ambiguous or technical terms.
4. Keep only what the speaker said or clearly implied.
5. State any interpretive assumptions separately.

## Output format

```markdown
## Charitable reconstruction
They are arguing that...

## Key terms
- ...

## Interpretive assumptions
- ...
```

## Quality checks

- The restatement should not be a straw man.
- The restatement should not add new evidence.
- The restatement should be stronger in clarity, not stronger by invention.

## Example

Input: `School should start later because students are tired in first period.`

Output: `They are arguing that the school should start later because some students are tired in first period.`
