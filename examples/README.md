# Examples

This folder contains example inputs and expected outputs for testing the argument reconstruction skills.

## Included examples

- `school-start-times/` — demonstrates the full workflow on the school-start-time argument from the skill pack.

## How developers can use these examples

1. Feed `input.md` into `argument_reconstruction_full/SKILL.md`.
2. Compare the model output with `expected_output.md`.
3. The wording does not need to match exactly, but the output should include the same core reconstruction moves:
   - charitable restatement
   - conclusion
   - explicit premises
   - hidden bridge
   - linkage
   - removal test
   - counterexample
   - fallacy/shortcut check
   - reality check
   - stronger version and disconfirming evidence
